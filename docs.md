# ProofLedger Documentation

## What is ProofLedger

ProofLedger anchors document hashes to Bitcoin via Stacks.
Every anchored document gets a permanent, verifiable proof of existence.

## How to Anchor a Document

1. Go to proofleger.vercel.app
2. Connect your Hiro wallet
3. Upload any file
4. Click Anchor Document
5. Approve the transaction in Hiro wallet
6. Your proof is now on Bitcoin

## Document Types

diploma, certificate, research, art, contribution, award, other

## Verification

Anyone can verify a document without a wallet.
Go to the Verify tab and upload the file.
ProofLedger recomputes the hash and checks the blockchain.

## Public Profiles
